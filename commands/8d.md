启动8D问题解决报告流程。

## 执行步骤

1. 先创建工作会话：
```bash
python3 scripts/session_manager.py --action create --skill 8d-report --project "$ARGUMENTS"
```

2. 读取 `skills/8d-report/SKILL.md` 获取完整8D工作流

3. 按8D八步法逐步引导用户：
   - D0: 准备工作 — 评估是否需要启动8D
   - D1: 成立团队 — 确认跨职能团队成员
   - D2: 描述问题 — 用5W2H方法定义问题
   - D3: 临时遏制措施 — 保护客户的紧急措施
   - D4: 根本原因分析 — 5Why/鱼骨图/FTA
   - D5: 制定永久纠正措施
   - D6: 实施并验证纠正措施
   - D7: 预防再发生 — 横向展开
   - D8: 团队认可与关闭

4. 每步完成后保存检查点：
```bash
python3 scripts/session_manager.py --action checkpoint \
  --session-id <session_id> --step "D2" \
  --data '{"problem": "..."}' \
  --next-hint "下一步: D3临时遏制"
```

5. 完成后生成8D报告文档：
```bash
python3 scripts/generate_quality_doc.py --type docx --template 8d_report --data <data.json> --output <output.docx>
```

## 参数

- $ARGUMENTS: 项目名称或问题描述，例如 "大众客诉-尺寸超差"

## 示例

```
/8d 大众客诉-尺寸超差
/8d 供应商A来料不良
```
