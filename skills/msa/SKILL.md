# MSA 测量系统分析

## 概述

MSA（Measurement System Analysis）是汽车行业五大核心工具之一，用于评估测量系统的统计特性，确保测量数据的质量。IATF 16949要求对控制计划中引用的测量系统进行MSA研究。本技能支持计量型和计数型MSA的规划、执行与结果评估。

## 何时触发

- 用户提到"MSA"、"测量系统分析"、"Gage R&R"、"GR&R"
- PPAP提交需要MSA研究
- 需要评估量具的重复性和再现性
- 测量数据质量问题排查

## 测量系统统计特性

### 五大特性
| 特性 | 定义 | 评估方法 |
|---|---|---|
| 偏倚(Bias) | 测量值与参考值的差异 | 偏倚分析 |
| 线性(Linearity) | 偏倚在量程范围内的变化 | 线性分析 |
| 稳定性(Stability) | 测量系统随时间的漂移 | 稳定性分析（控制图） |
| 重复性(Repeatability) | 同一操作者多次测量的变异(EV) | Gage R&R |
| 再现性(Reproducibility) | 不同操作者测量的变异(AV) | Gage R&R |

## 计量型MSA — Gage R&R研究

### 交叉Gage R&R（Crossed）
**适用条件**：每位操作者测量每个零件（最常用方法）

**标准方案**：
- 零件数：10个（代表过程变异）
- 操作者数：3人（代表实际测量人员）
- 重复次数：3次
- 总测量数：10 × 3 × 3 = 90

**结果判定标准（AIAG第4版）**：
| %GRR | 过程分类 | 接受性 |
|---|---|---|
| <10% | 测量系统优秀 | 可接受 |
| 10%-30% | 测量系统可接受 | 视应用而定（特殊特性不可接受） |
| >30% | 测量系统不可接受 | 必须改进 |

**ndc（区别分类数）要求**：ndc ≥ 5
- ndc = 1.41 × (σ_part / σ_gage)

### 嵌套Gage R&R（Nested）
**适用条件**：破坏性测试（每个零件只能测一次）

### 方差分析(ANOVA)方法
```
总变异 = 零件间变异 + 操作者变异 + 交互作用 + 重复性
%GRR = 100 × (σ_gage / σ_total)
σ_gage = √(σ_repeatability² + σ_reproducibility²)
```

## 计数型MSA

### 属性一致性分析法（Attribute Agreement Analysis）
**适用场景**：通/止规、目视检查等计数型测量

**评估指标**：
- 操作者自身一致性（重复性）
- 操作者间一致性（再现性）
- 操作者与标准一致性（准确性）
- Kappa值：≥0.75为良好，≥0.90为优秀

### 风险分析法（Risk Analysis Method）
**适用场景**：无法获取参考标准值时

**评估维度**：
- 误判率（将合格判为不合格）
- 漏判率（将不合格判为合格）

## 偏倚与线性分析

### 偏倚分析
1. 选择参考标准件（已知参考值）
2. 测量≥10次
3. 计算均值偏倚 = X̄ - 参考值
4. t检验判断偏倚是否显著

### 线性分析
1. 选择5个以上覆盖量程的参考标准件
2. 每个标准件重复测量≥10次
3. 计算各点偏倚
4. 线性回归分析偏倚与参考值的关系
