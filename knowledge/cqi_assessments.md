# CQI特殊过程评估概述

## 概述

CQI（Continuous Quality Improvement）特殊过程评估是AIAG发布的行业评估标准，针对汽车制造中无法通过后续检验完全验证的过程质量。IATF 16949条款8.5.1.3要求对这些特殊过程进行定期评估。

## 评估清单

### CQI-8: 分层过程审核(LPA)
- **评估对象**：制造过程的分层审核体系
- **审核层级**：三级（操作层/监督层/管理层）
- **审核频率**：每日/每周/每月
- **关键检查项**：
  - 操作员是否遵循作业指导书
  - 过程参数是否在规定范围内
  - 不合格品是否正确标识和隔离
  - 防错装置是否正常工作

### CQI-9: 热处理系统评估
- **评估对象**：热处理过程（淬火、回火、渗碳、氮化等）
- **评估维度**：
  - 质量体系要求
  - 设备和控制
  - 过程控制
  - 作业和文件
- **关键检查项**：
  - 温度均匀性测试(TUS)
  - 系统精度测试(SAT)
  - 热电偶校准
  - 碳势控制
  - 淬火介质管理

### CQI-11: 电镀系统评估
- **评估对象**：电镀和化学镀过程
- **关键检查项**：
  - 镀液化学成分控制
  - 电流密度控制
  - 前处理工艺控制
  - 镀层厚度测量
  - 盐雾试验管理
  - 废水处理合规

### CQI-12: 涂装系统评估
- **评估对象**：喷涂、浸涂、电泳等涂装过程
- **关键检查项**：
  - 涂料配比控制
  - 膜厚测量
  - 烘烤温度/时间控制
  - 前处理管理
  - 附着力/硬度测试
  - VOC排放合规

### CQI-15: 焊接系统评估
- **评估对象**：电阻焊、弧焊、激光焊等焊接过程
- **关键检查项**：
  - 焊接参数监控（电流、电压、时间、压力）
  - 焊接设备维护
  - 电极/焊帽管理
  - 焊接质量检验（撕裂试验、金相、超声波）
  - 焊接操作员资质
  - 焊接规程(WPS/WPQR)

### CQI-17: 钎焊系统评估
- **评估对象**：软钎焊、硬钎焊过程
- **关键检查项**：
  - 钎料和钎剂管理
  - 温度控制
  - 钎焊间隙控制
  - 钎焊质量检验
  - 操作员资质

### CQI-23: 模塑系统评估
- **评估对象**：注塑、吹塑、压塑等模塑过程
- **关键检查项**：
  - 模具温度控制
  - 注塑参数监控
  - 材料干燥管理
  - 模具维护
  - 外观检验标准
  - 尺寸控制

### CQI-27: 铸造系统评估
- **评估对象**：压铸、砂铸、精密铸造等铸造过程
- **关键检查项**：
  - 熔炼温度控制
  - 模具/型芯管理
  - 铸造参数监控
  - X射线/CT检测
  - 热处理控制
  - 缺陷分级标准

## 评估流程

### 评估前准备
1. 确定评估范围和适用CQI标准
2. 组建评估团队（至少1名CQI认证评估师）
3. 收集相关文件和记录
4. 安排现场评估时间

### 现场评估
1. 按CQI检查表逐项评估
2. 评分：符合/部分符合/不符合
3. 记录证据和观察
4. 现场确认纠正措施需求

### 评估报告
1. 汇总评分结果
2. 列出不符合项
3. 制定纠正措施计划
4. 跟踪验证关闭

## 评估周期建议

| 评估等级 | 建议频率 | 触发条件 |
|---|---|---|
| 初次评估 | 新过程/新供应商 | 过程启动前 |
| 定期评估 | 每12-24个月 | 正常周期 |
| 追加评估 | 发生重大质量问题 | 客户投诉/退货 |
