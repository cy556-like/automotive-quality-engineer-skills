# 汽车行业五大核心工具概述

## 概述

IATF 16949要求汽车行业供应商掌握并应用五大核心质量工具。这五个工具相互关联，贯穿产品从策划到量产的全过程，构成了汽车行业质量管理体系的技术基础。

## 工具关联图

```
APQP（先期产品质量策划）
├── 阶段2 → DFMEA
├── 阶段3 → PFMEA
├── 阶段3 → 控制计划（试生产）
├── 阶段3 → MSA计划
├── 阶段3 → SPC计划
├── 阶段4 → MSA执行
├── 阶段4 → SPC执行（初始过程能力）
├── 阶段4 → PPAP提交
└── 阶段5 → SPC持续监控
```

## 1. APQP — 先期产品质量策划

### 参考手册
- AIAG APQP手册（第2版，2008年）
- AIAG APQP手册（第3版，2024年更新）

### 五个阶段
| 阶段 | 名称 | 关键里程碑 |
|---|---|---|
| 1 | 计划和确定项目 | 项目立项批准 |
| 2 | 产品设计和开发 | 设计评审通过 |
| 3 | 过程设计和开发 | 过程评审通过 |
| 4 | 产品和过程确认 | PPAP批准 |
| 5 | 反馈、评定和纠正措施 | 持续改进 |

### 关键时间节点
- 阶段1→2：概念批准
- 阶段2→3：设计批准
- 阶段3→4：过程批准
- 阶段4→5：量产批准

## 2. FMEA — 失效模式与影响分析

### 参考手册
- AIAG-VDA FMEA手册（第1版，2019年）— 协调版

### FMEA类型
| 类型 | 缩写 | 应用阶段 | 分析对象 |
|---|---|---|---|
| 设计FMEA | DFMEA | 产品设计阶段 | 产品设计 |
| 过程FMEA | PFMEA | 过程设计阶段 | 制造过程 |
| 监控和系统响应FMEA | FMEA-MSR | 量产阶段 | 监控系统 |

### 七步法概览
1. 策划与准备 → 定义范围和团队
2. 结构分析 → 建立分析层级
3. 功能分析 → 定义功能和要求
4. 失效分析 → 识别失效链路
5. 风险分析 → 评估S-O-D和AP
6. 优化 → 制定措施降低风险
7. 结果文件化 → 形成FMEA报告

### 评分标准（1-10分制）

#### 严重度(S)评估
| 评分 | DFMEA | PFMEA |
|---|---|---|
| 10 | 潜在失效影响符合安全/法规 | 潜在失效影响符合安全/法规 |
| 9 | 潜在失效影响符合安全/法规（有警告） | 潜在失效影响符合安全/法规（有警告） |
| 8 | 主要功能丧失 | 主要功能丧失 |
| 7 | 主要功能退化 | 主要功能退化 |
| 6 | 舒适/便利功能丧失 | 舒适/便利功能丧失 |
| 5 | 舒适/便利功能退化 | 舒适/便利功能退化 |
| 4 | 外观/噪音缺陷（多数客户发现） | 外观/噪音缺陷（多数客户发现） |
| 3 | 外观/噪音缺陷（少数客户发现） | 外观/噪音缺陷（少数客户发现） |
| 2 | 外观/噪音缺陷（极少数客户发现） | 外观/噪音缺陷（极少数客户发现） |
| 1 | 无可辨识的影响 | 无可辨识的影响 |

## 3. SPC — 统计过程控制

### 参考手册
- AIAG SPC手册（第2版）
- AIAG-VDA SPC手册（新黄皮书）

### 过程能力指数标准
| 指标 | 公式 | PPAP要求 |
|---|---|---|
| Cp | (USL-LSL)/(6σ_within) | ≥1.33 |
| Cpk | min[(USL-X̄)/(3σ_within), (X̄-LSL)/(3σ_within)] | ≥1.33（特殊特性）|
| Pp | (USL-LSL)/(6σ_overall) | ≥1.67 |
| Ppk | min[(USL-X̄)/(3σ_overall), (X̄-LSL)/(3σ_overall)] | ≥1.67（特殊特性）|

### 控制图选择决策树
```
数据类型？
├── 计量型
│   ├── 子组大小=1 → I-MR图
│   ├── 子组大小2-9 → Xbar-R图
│   └── 子组大小≥10 → Xbar-S图
└── 计数型
    ├── 不合格品率 → P图（子组可变）或NP图（子组固定）
    └── 缺陷数 → C图（子组固定）或U图（子组可变）
```

## 4. MSA — 测量系统分析

### 参考手册
- AIAG MSA手册（第4版）

### Gage R&R判定标准
| %GRR | 判定 | 说明 |
|---|---|---|
| <10% | 可接受 | 测量系统良好 |
| 10%-30% | 有条件接受 | 视应用而定；特殊特性不可接受 |
| >30% | 不可接受 | 必须改进测量系统 |

### ndc要求
- ndc（区别分类数）≥ 5
- ndc = 1.41 × (σ_part / σ_gage)

## 5. PPAP — 生产件批准程序

### 参考手册
- AIAG PPAP手册（第4版）

### 18要素清单
1. 设计记录
2. 工程更改文件
3. 客户工程批准（如需要）
4. 设计FMEA
5. 过程流程图
6. 过程FMEA
7. 控制计划
8. 测量系统分析研究
9. 尺寸结果
10. 材料/性能试验结果
11. 初始过程研究
12. 合格实验室文件
13. 外观批准报告(AAR)
14. 样品生产件
15. 主样品
16. 检具和试验设备
17. 客户特殊要求
18. 零件提交保证书(PSW)

### 提交等级
- Level 1：仅PSW
- Level 2：PSW + 样品 + 部分支持文件
- Level 3：PSW + 样品 + 完整支持文件（默认）
- Level 4：PSW + 其他客户要求
- Level 5：PSW + 样品 + 完整支持文件 + 现场评审
